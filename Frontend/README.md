# Frontend

This folder contains a small static frontend for the notification system.

## Usage
1. Start the backend API from `backend/`.
2. Open `Frontend/index.html` in your browser.
3. Use the login form to create a user and retrieve a token.
4. Use the UI to create, refresh, mark read, and delete notifications.

## Notes
- The frontend expects the backend API at `http://localhost:3000/api/v1`.
- No build step is required; this is a static HTML + JavaScript app.
